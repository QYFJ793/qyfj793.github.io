Rust 语言之字符串

简介

在 Rust 中，字符串是文本数据的表示方式。Rust 提供了两种主要的字符串类型：String 和 &str。String 是一个可增长的、可变、拥有所有权的字符串类型，而 &str 是一个不可变的字符串切片，它借用了其他数据的所有权。

String 类型

String 是一个可增长的字符串类型，它在堆上分配内存。你可以使用 + 运算符或者 push_str 方法来追加字符串。
